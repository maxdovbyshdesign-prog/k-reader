# Ksyusha Reader 🌸

> **Твоя личная говорящая библиотека** — уютное, теплое и полностью приватное приложение-аудиоридер для мобильных телефонов и компьютеров, созданное для комфортного чтения и прослушивания книг.

Приложение позволяет загружать электронные книги популярных форматов, выбирать голос озвучки, настраивать скорость чтения, сохранять закладки с заметками и продолжать чтение ровно с того места, где вы остановились. Особенный фокус сделан на естественное воспроизведение текстов на русском языке.

---

## ✨ Что умеет Ksyusha Reader (MVP)

1. **Загрузка и чтение популярных форматов**:
   - `TXT` — Простой текст, автоматически разделяемый по смысловым частям.
   - `EPUB` — Стандартные файлы электронных книг с полноценным извлечением оригинальных глав.
   - `FB2` — Фирменный российский XML-формат книг с точным разбором глав и абзацев.
   - `PDF` — Документы с текстовым слоем. Присутствует автоматическое обнаружение сканированных книг (картинок) с выводом предупреждения о том, что OCR пока не поддерживается.
   - `DOCX` — Файлы документов Microsoft Word.

2. **Два встроенных режима озвучки (Браузерные и статические)**:
   - **Системный голос (Web Speech API)**: работает полностью **офлайн из коробки** на любом устройстве (Android-смартфоны, iOS, Windows, macOS). Использует качественные локальные голоса операционной системы (например, "Google русский" на Android). Поддерживает русский, английский и иврит.
   - **Piper Web Neural Voice (Экспериментальный)**: высококачественный нейросетевой голос, работающий 100% прямо внутри браузера на базе технологии ONNX Runtime Web. Файлы голоса скачиваются один раз и сохраняются в кэше браузера навсегда через **Cache Storage API**.

3. **Колыбельный и дорожный интерфейс (Car-friendly UX)**:
   - Крупные, контрастные кнопки плеера (Play, Пауза, Стоп, Шаг вперед/назад), в которые легко попадать пальцем во время вождения автомобиля или приготовления еды на кухне.
   - Удобная навигационная полоса по абзацам главы.

4. **Интеллектуальное ведение фокуса**:
   - Читаемый абзац подсвечивается теплым фоном и отображается крупным шрифтом в центре экрана.
   - Предыдущие и последующие абзацы плавно приглушены по контрасту, помогая следить за мыслью.
   - Клик по любому абзацу мгновенно переносит плеер на это предложение.

5. **Закладки и заметки**:
   - Создавайте закладки во время прослушивания с вашим собственным комментарием.
   - Переходите к сохраненным закладкам в один клик.

6. **Абсолютная приватность и Offline-first**:
   - Приложение **не требует регистрации, логинов и интернета** для базовой работы.
   - Книги, история чтения, закладки и выбранные настройки хранятся в защищенной памяти вашего браузера (через стандартную базу данных **IndexedDB**). Ваши файлы не отправляются на удаленные облака.

## 🚀 Как запустить приложение локально

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер разработки Vite:
   ```bash
   npm run dev
   ```
3. Откройте адрес `http://localhost:3000` в браузере вашего компьютера или телефона.

## 🎤 Настройка и деплой нейросетевых голосов Piper на GitHub Pages

Приложение Ksyusha Reader является полностью статическим сайтом. Чтобы нейросетевые голоса **Piper Web Neural Voice** работали из коробки у всех ваших читателей, вы можете загрузить файлы моделей прямо в ваш репозиторий.

### Шаг 1. Структура папок в вашем репозитории
Для автоматического обнаружения моделей разместите файлы в корневой директории `public/models/piper/`:
```text
📁 public/
 └── 📁 models/
      └── 📁 piper/
           ├── 📄 ru_RU-dmitry-medium.onnx
           ├── 📄 ru_RU-dmitry-medium.json
           ├── 📄 ru_RU-irina-medium.onnx
           └── 📄 ru_RU-irina-medium.json
```

### Шаг 2. Где взять файлы голосов
Вы можете скачать готовые натренированные голоса напрямую из официального хранилища:
1. Перейдите на [HuggingFace Piper Voices](https://huggingface.co/rhasspy/piper-voices/tree/main).
2. Зайдите в папку нужного языка (например, `ru_RU` для русского, `en_US` для английского, `he_IL` для иврита).
3. Скачайте файл модели `.onnx` (например, `ru_RU-dmitry-medium.onnx`) и сопутствующий конфигурационный файл `.json` (например, `ru_RU-dmitry-medium.json`).
4. Поместите их в папку `public/models/piper/` вашего проекта.

### Шаг 3. Деплой на GitHub Pages
После отправки изменений в ваш репозиторий на GitHub:
1. Зайдите в настройки репозитория (**Settings -> Pages**).
2. Выберите ветку для публикации (обычно `main` или `master`) и папку (обычно `/ (root)`).
3. Сохраните. Сайт станет доступен по адресу `https://ваш-логин.github.io/название-репозитория/`.
4. Встроенный менеджер путей автоматически адаптирует ссылки, поэтому Piper и его кэширование будут работать идеально на любом поддомене GitHub!

## 📂 Структура проекта

- `src/types.ts` — Общие контракты данных, интерфейсы настроек, книг и TTS-провайдеров.
- `src/lib/db.ts` — Обёртка работы с базой данных IndexedDB. Обеспечивает сохранение книг, истории чтения, настроек и закладок.
- `src/lib/parsers.ts` — Универсальные клиентские парсеры форматов TXT, EPUB, FB2, PDF (через `pdfjs-dist`) и DOCX (через `mammoth`).
- `src/lib/tts.ts` — Провайдеры Web Speech API (со встроенной навигацией) и Silero API Client.
- `src/components/CozyHeader.tsx` — Панель навигации ридера в фирменном стиле.
- `src/components/LibraryView.tsx` — Книжная полка, прогресс-бары, функционал Drag-and-Drop и парсинг.
- `src/components/ReaderView.tsx` — Центральный аудиоэкран ридера с фокусом на абзацы и расширенным пультом управления.
- `src/components/SettingsView.tsx` — Панель выбора озвучки, тестирования соединений, тем оформления и текстовой инструкцией.
- `src/components/AboutView.tsx` — Уютный раздел с описанием и полезными советами по чтению.
